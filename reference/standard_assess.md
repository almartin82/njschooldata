# call the correct `fetch` function for normal assessment years

for 2008-2014, this function will grab the NJASK for gr 3-8, and HSPA
for grade 11

## Usage

``` r
standard_assess(end_year, grade)
```

## Arguments

- end_year:

  a school year. end_year is the end of the academic year - eg 2013-14
  school year is end_year '2014'. valid values are 2004-2014.

- grade:

  a grade level. valid values are 3,4,5,6,7,8
