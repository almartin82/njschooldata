# Get cache statistics and information

Returns information about the current cache state including number of
items, memory usage, and hit/miss statistics.

## Usage

``` r
njsd_cache_info()
```

## Value

A list with cache information

## Examples

``` r
njsd_cache_info()
#> njschooldata Session Cache
#> ==========================
#> Status:     Enabled
#> Items:      0
#> Memory:     0.00 MB
#> Cache hits: 0
#> Misses:     0
```
