# Check if year is valid for a data type

Check if year is valid for a data type

## Usage

``` r
is_valid_year(end_year, data_type)
```

## Arguments

- end_year:

  The school year (end year)

- data_type:

  One of "enrollment", "parcc", "grate_4yr", "grate_5yr", "gcount",
  "legacy_assess"

## Value

Logical indicating if year is valid
