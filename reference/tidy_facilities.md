# Tidy processed facilities rows into the canonical long schema

Tidy processed facilities rows into the canonical long schema

## Usage

``` r
tidy_facilities(processed_list)
```

## Arguments

- processed_list:

  List of processed facilities long frames.

## Value

Data frame with
[`facilities_columns()`](https://almartin82.github.io/njschooldata/reference/facilities_columns.md).
