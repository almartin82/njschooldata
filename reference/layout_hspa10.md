# HSPA 2010 Fixed-Width File Layout

Column layout specification for reading High School Proficiency
Assessment (HSPA) 2010 fixed-width format data files.

A processed R version of the 'File Layout' for the 2004-2010 HSPA.
Original file is in .xls format on NJDOE website

## Usage

``` r
layout_hspa10

layout_hspa10
```

## Format

A data frame with column layout specifications including field
positions, lengths, data types, descriptions, and final column names.

- field_start_position:

  field_start_position

- field_end_position:

  field_end_position

- field_length:

  field_length

- data_type:

  data_type

- description:

  description

- comments:

  comments

- valid_values:

  valid_values

- spanner1:

  spanner1

- spanner2:

  spanner2

- final_name:

  final_name

## Source

NJ Department of Education HSPA 2010 file specifications

NJDOE website - eg the 'File Layout' link on
