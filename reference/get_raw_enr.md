# Read a zipped Excel fall enrollment file from the NJ state website

Read a zipped Excel fall enrollment file from the NJ state website

## Usage

``` r
get_raw_enr(end_year)
```

## Arguments

- end_year:

  A school year. Year is the end of the academic year - eg 2006-07
  school year is year '2007'. Valid values are 2000-2025.

## Value

Data frame with raw enrollment data
