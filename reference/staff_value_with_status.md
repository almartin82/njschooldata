# Pair staff numeric coercion with value-status classification

Pair staff numeric coercion with value-status classification

## Usage

``` r
staff_value_with_status(x)
```

## Arguments

- x:

  A raw staff value vector.

## Value

A list with `value` from
[`staff_value_numeric`](https://almartin82.github.io/njschooldata/reference/staff_value_numeric.md)
and `status` from
[`classify_value_status`](https://almartin82.github.io/njschooldata/reference/classify_value_status.md).
