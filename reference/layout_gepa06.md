# GEPA 2006 Fixed-Width File Layout

Column layout specification for reading Grade Eight Proficiency
Assessment (GEPA) 2006 fixed-width format data files.

## Usage

``` r
layout_gepa06
```

## Format

A data frame with column layout specifications including field
positions, lengths, data types, descriptions, and final column names.

## Source

NJ Department of Education GEPA 2006 file specifications
