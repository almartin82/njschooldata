# Validate an SGP request year, with a type-specific explanation

Validate an SGP request year, with a type-specific explanation

## Usage

``` r
sgp_check_year(end_year, type)
```

## Arguments

- end_year:

  School year end.

- type:

  SGP measure type.

## Value

Invisibly `NULL`; [`stop()`](https://rdrr.io/r/base/stop.html)s for
unsupported combinations.
