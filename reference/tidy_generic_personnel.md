# tidy generic personnel indicator data frame

tidy generic personnel indicator data frame

## Usage

``` r
tidy_generic_personnel(df, end_year, indicator)
```

## Arguments

- df:

  personnel data frame, eg output of get_raw_tges() indicators 16-19

- end_year:

  end year that the report was published

- indicator:

  character, indicator name

## Value

long, tidy data frame
