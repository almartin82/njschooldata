# Tidy the State by Ed Environ sheet (5-21 or 3-5)

Tidy the State by Ed Environ sheet (5-21 or 3-5)

## Usage

``` r
tidy_sped_placement_state(df, age_group)
```

## Arguments

- df:

  raw tibble from `get_raw_sped_placement(level = "state")`

- age_group:

  "5-21" or "3-5"

## Value

tidy tibble in the same long shape as the district output
