# Map Sheet Names Across Years

This list maps common sheet name variations. Format is:
`list(canonical_name = list(year_range = "actual_sheet_name"))`

## Usage

``` r
spr_sheet_mapping
```

## Format

An object of class `list` of length 2.

## Details

Internal data structure mapping sheet name variations across years. Some
SPR sheet names changed over time (e.g., 2018-2019 vs 2020+).
