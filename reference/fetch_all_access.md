# Fetch all ACCESS for ELLs results

Convenience function to download and combine all available ACCESS for
ELLs results into a single data frame.

## Usage

``` r
fetch_all_access()
```

## Value

A data frame with all ACCESS results (2022-2024, all grades)

## Examples

``` r
if (FALSE) { # \dontrun{
# Get all ACCESS results (takes a while)
all_access <- fetch_all_access()
} # }
```
