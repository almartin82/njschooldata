# Harmonize the AP/IB/dual participation-by-group sheet across the schema drift

Harmonize the AP/IB/dual participation-by-group sheet across the schema
drift

## Usage

``` r
.advanced_participation_by_group(end_year, level, coerce_values = TRUE)
```

## Arguments

- end_year:

  School year end.

- level:

  One of `"school"` or `"district"`.

- coerce_values:

  Whether to coerce published value tokens to numeric.
