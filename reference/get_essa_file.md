# Get an ESSA comprehensive or targeted accountability file

Get an ESSA comprehensive or targeted accountability file

## Usage

``` r
get_essa_file(end_year, file_type = "comprehensive")
```

## Arguments

- end_year:

  a school year. end_year is the end of the academic year - eg 2016-17
  school year is end_year 2017. valid values are 2017

- file_type:

  'comprehensive' or 'targeted'?

## Value

list of data frames
