# HSPA 2004 Fixed-Width File Layout

Column layout specification for reading High School Proficiency
Assessment (HSPA) 2004 fixed-width format data files.

## Usage

``` r
layout_hspa04
```

## Format

A data frame with column layout specifications including field
positions, lengths, data types, descriptions, and final column names.

## Source

NJ Department of Education HSPA 2004 file specifications
