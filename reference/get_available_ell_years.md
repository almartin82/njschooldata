# Get the years for which NJ English Learner population data is available

Get the years for which NJ English Learner population data is available

## Usage

``` r
get_available_ell_years()
```

## Value

integer vector of valid \`end_year\` values

## Examples

``` r
get_available_ell_years()
#>  [1] 2006 2007 2008 2009 2010 2011 2012 2013 2014 2015 2016 2017 2018 2019 2020
#> [16] 2021 2022 2023 2024 2025 2026
```
