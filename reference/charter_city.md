# Charter School Host District Mapping

A dataset containing charter schools and their host districts/cities in
New Jersey. Charter schools are public schools that operate
independently but are accountable to their authorizing districts (host
districts).

## Usage

``` r
charter_city
```

## Format

A data frame with 137 rows and 6 columns:

- district_id:

  Charter school district identifier

- district_name:

  Charter school district name

- host_county_id:

  County code of the host district

- host_county_name:

  County name of the host district

- host_district_id:

  Host district identifier

- host_district_name:

  Host district name

## Source

NJ Department of Education
