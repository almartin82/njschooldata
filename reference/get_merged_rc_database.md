# Combines school and district Performance Reports for 2017-on, when two files were released.

Combines school and district Performance Reports for 2017-on, when two
files were released.

## Usage

``` r
get_merged_rc_database(end_year)
```

## Arguments

- end_year:

  end of the academic year. Valid values are 2017, 2018, 2019

## Value

list of dataframes
