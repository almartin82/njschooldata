# NJASK 2007 Grade 3 Fixed-Width File Layout

Column layout specification for reading New Jersey Assessment of Skills
and Knowledge (NJASK) 2007 Grade 3 fixed-width format data files.

A processed R version of the 'File Layout' for the 2007 and 2008 NJASK,
elementary (gr 3 and 4) version. Original file is in .xls format on
NJDOE website

## Usage

``` r
layout_njask07gr3

layout_njask07gr3
```

## Format

A data frame with column layout specifications including field
positions, lengths, data types, descriptions, and final column names.

- field_start_position:

  field_start_position

- field_end_position:

  field_end_position

- field_length:

  field_length

- data_type:

  data_type

- description:

  description

- comments:

  comments

- valid_values:

  valid_values

- spanner1:

  spanner1

- spanner2:

  spanner2

- final_name:

  final_name
