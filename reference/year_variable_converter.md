# year variable converter

for the 1999-2003 tges files, the 'year' of the data was encoded in the
variable names

## Usage

``` r
year_variable_converter(df, end_year)
```

## Arguments

- df:

  a tges indicator data frame published between 1999 and 2003

- end_year:

  year published

## Value

data frame that conforms to 2004-2009 style
