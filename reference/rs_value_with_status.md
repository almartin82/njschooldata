# Pair DARS numeric coercion with value-status classification

Pair DARS numeric coercion with value-status classification

## Usage

``` r
rs_value_with_status(x)
```

## Arguments

- x:

  A raw DARS value vector.

## Value

A list with `value` from
[`rs_value_numeric`](https://almartin82.github.io/njschooldata/reference/rs_value_numeric.md)
and `status` from
[`classify_value_status`](https://almartin82.github.io/njschooldata/reference/classify_value_status.md).
