# Get Raw Taxpayer's Guide to Educational Spending

Downloads and unpacks one year of the NJ DOE Taxpayers' Guide to
Educational Spending (TGES; branded the "Comparative Spending Guide" /
CSG before 2011). Each year is published as a single zip.

## Usage

``` r
get_raw_tges(end_year)
```

## Arguments

- end_year:

  a school year. end_year is the end of the academic year - eg the
  2016-17 school year is end_year 2017. Valid values are 2001-2025. (NJ
  DOE links 1999 and 2000 but those downloads 404 on the state site.)

## Value

list of data frames
