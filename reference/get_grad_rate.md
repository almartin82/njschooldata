# Get NJ graduation rate data

Get NJ graduation rate data

## Usage

``` r
get_grad_rate(end_year, methodology)
```

## Arguments

- end_year:

  End of the academic year - 2011-2012 is 2012. Valid values are
  2011-2024.

- methodology:

  Character string specifying calculation methodology. One of "4 year"
  or "5 year".

## Value

dataframe with the number of graduates per school and district
