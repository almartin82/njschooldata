# Enrich matriculation rates with counts from grad counts

Enrich matriculation rates with counts from grad counts

## Usage

``` r
enrich_matric_counts(df, type = "16 month")
```

## Arguments

- df:

  data frame of including matriculation percentages

- type:

  character string specifying matriculation type. One of "16 month" or
  "12 month"

## Value

data_frame
