# Tidy Budgeted vs Actual Fund Balance

Tidy Budgeted vs Actual Fund Balance

## Usage

``` r
tidy_budgeted_vs_actual_fund_balance(df, end_year)
```

## Arguments

- df:

  general fund vs actual used data frame, eg CSG20 output from
  get_raw_tges()

- end_year:

  end year that the report was published

## Value

data frame
