# NJASK 2007 Grade 5 Fixed-Width File Layout

Column layout specification for reading New Jersey Assessment of Skills
and Knowledge (NJASK) 2007 Grade 5 fixed-width format data files.

## Usage

``` r
layout_njask07gr5
```

## Format

A data frame with column layout specifications including field
positions, lengths, data types, descriptions, and final column names.

## Source

NJ Department of Education NJASK 2007 Grade 5 file specifications
