# Tidy Grad Count

Transforms graduation count data to long format.

## Usage

``` r
tidy_grad_count(df, end_year)
```

## Arguments

- df:

  Output of process_grad_count

- end_year:

  End of the academic year - eg 2006-07 is 2007. Valid values are
  1998-present.

## Value

data.frame with number of graduates
