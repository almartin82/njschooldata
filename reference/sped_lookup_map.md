# Special Education District Lookup Map

Mapping of county names to district names and district IDs for New
Jersey school districts. Used for matching special education data to
districts.

## Usage

``` r
sped_lookup_map
```

## Format

A data frame with 578 rows and 3 columns:

- county_name:

  County name

- district_name:

  District name

- district_id:

  District identifier code

## Source

NJ Department of Education
