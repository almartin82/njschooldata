# tidy common/generic budget indicator data frame

tidy common/generic budget indicator data frame

## Usage

``` r
tidy_generic_budget_indicator(df, end_year, indicator)
```

## Arguments

- df:

  indicator data frame, eg output of get_raw_tges() indicators 1-15

- end_year:

  end year that the report was published

- indicator:

  character, indicator name

## Value

long, tidy data frame
