Enrollment data valid years Note: 1999 (1998-99 school year) is the

Comment in this file used to claim 1999 was removed, but the ZIP is
still served and now parses through process_enr() after the
split_enr_cols() ordering fix.

## Usage

``` r
ENR_VALID_YEARS
```

## Format

An object of class `integer` of length 28.
