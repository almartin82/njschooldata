# njschooldata

[![R-CMD-check](https://github.com/almartin82/njschooldata/actions/workflows/R-CMD-check.yaml/badge.svg)](https://github.com/almartin82/njschooldata/actions/workflows/R-CMD-check.yaml)

Fetch and analyze New Jersey public school enrollment data from NJDOE.

## Installation

```r
# install.packages("remotes")
remotes::install_github("almartin82/njschooldata")
```

## Part of the 50 State Schooldata Family

This package is part of a family of R packages providing school enrollment data for all 50 US states.

**All packages:** [github.com/almartin82](https://github.com/almartin82?tab=repositories&q=schooldata)
